# AI 简历优化 & JD匹配 Agent

一个基于 **CrewAI** 的多Agent系统 + Streamlit 网页Demo。

## 功能
- 上传简历 PDF
- 粘贴职位描述 (JD)
- 自动完成：JD解析 → 简历分析 → 匹配度评估 → 简历优化 → Cover Letter 生成

## 技术栈
- CrewAI（多Agent框架）
- Groq（免费 LLM）
- Streamlit（网页界面）
- PyPDF（PDF解析）

## 如何本地运行
```bash
uv run streamlit run app.py
